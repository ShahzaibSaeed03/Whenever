Common questions about uploading your work securely.

What file types are accepted?

We accept .zip, .docx, .jpg, .png, and similar formats. We do not accept .exe or .js files.

What is the maximum file size?

Each file must be under 120 MB. Please compress your file if needed before uploading.

Can I edit the uploaded file later?

No. To maintain legal integrity, uploaded files and their metadata cannot be modified afterward.

Why should I include drafts?

Uploading drafts shows the creative process and strengthens proof of originality.
